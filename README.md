# Plate_recognizing Локализация автомобильных номеров (License Plate Bounding Box) на PyTorch

## Обзор проекта

В данном проекте реализована легковесная сверточная нейронная сеть (CNN)
для автоматической локализации автомобильных номерных знаков на
изображениях.

Модель предсказывает ограничивающий прямоугольник (bounding box) в
нормализованном формате `(cx, cy, w, h)` и оптимизируется с
использованием IoU-ориентированных функций потерь.

Был использован датасет - https://www.kaggle.com/datasets/ronakgohil/license-plate-dataset

Система разработана с учетом устойчивости к сложным условиям съемки: -
туман - блики - ночная съемка - низкая контрастность

------------------------------------------------------------------------

## Архитектура модели

Основная модель (`PlateCenterSizeCNN`) включает:

-   Сверточный backbone (Conv + BatchNorm + ReLU)
-   Heatmap-голову для предсказания центра объекта (SoftArgmax)
-   Взвешенный pooling для регрессии ширины и высоты
-   Sigmoid-нормализацию выходных координат

Формат выхода:

\[cx, cy, w, h\] --- нормализованные координаты в диапазоне \[0, 1\]

------------------------------------------------------------------------

## Стратегия обучения

Обучение проводится в два этапа.

### Этап A --- Обучение признаков

-   Включены аугментации
-   Постепенное уменьшение sigma для heatmap
-   Learning rate: 3e-4
-   CosineAnnealing scheduler

### Этап B --- Тонкая настройка

-   Аугментации отключены
-   Warmup learning rate (первые 5 эпох)
-   Финальный learning rate: 2e-5
-   Фокус на улучшении IoU@0.5

------------------------------------------------------------------------

## Функции потерь

Итоговая функция потерь состоит из:

-   CIoU Loss (основная регрессия bbox)
-   L1 Loss (стабилизация)
-   MSE Loss для heatmap (контроль центра)

Основные метрики оценки:

-   IoU@0.5
-   IoU@0.7
-   Mean IoU (mIoU)

## Режимы инференса

### Базовый режим

Однопроходное предсказание bounding box.

### Двухэтапный режим (рекомендуется)

1.  Первичное предсказание на полном изображении
2.  Расширение ROI вокруг бокса
3.  Предобработка (CLAHE + grayscale)
4.  Повторное предсказание
5.  Выбор лучшего результата по confidence

Этот режим повышает устойчивость к плохой видимости.
